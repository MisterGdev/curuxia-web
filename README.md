# 🦉 Curuxia | Digitalización e Ingeniería IA

Digitalizamos hoy la empresa del mañana. Proyecto web corporativo centrado en la optimización de procesos mediante infraestructura IT y soluciones de Inteligencia Artificial.

## 🚀 Características
- **Arquitectura SPA**: Navegación fluida sin recarga de página.
- **Diseño Monumental**: Estética técnica basada en ingeniería y limpieza visual.
- **Mobile First**: Adaptado a cualquier dispositivo.
- **Interacción Dinámica**: Sistema de Q&A interactivo y formularios de captación.

## 📁 Estructura del Repositorio
- `/assets/css`: Estilos modulares y variables de marca.
- `/assets/js`: Lógica de navegación y componentes interactivos.
- `/assets/img`: Recursos visuales y logotipos.
